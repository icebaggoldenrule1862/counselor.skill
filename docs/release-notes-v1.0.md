# Release Notes

## v1.0.0

### 中文

`辅导员.skill` 的首个公开版本。

这一版的重点不是题材扩张，而是把“单一辅导员系统”真正做成型：

- 支持从主观描述、私聊、通知、约谈转写、制度文本中蒸馏辅导员人格
- 支持开口、追问、办公室约谈、说明文、补救、求助、复盘等多种模式
- 支持 correction、session、archive、snapshot 等长期层
- 支持现实同步路径：先预演，再把真实反馈带回来
- 提供完整系统样例、导入示例、蒸馏样例和本地运行脚本

这一版明确不做多角色群像，而是只做一个高保真、可长期使用的单一辅导员系统。

### English

The first public release of `counselor.skill`.

This release focuses on making the single-counselor system feel complete:

- distill counselor behavior from manual notes, private chats, notices, office-talk transcripts, and policy text
- support multiple communication modes including opening, follow-up, office talk, statement writing, repair, help-seeking, and debrief
- include long-term layers such as correction, session, archive, and snapshot
- support reality-sync workflows: rehearse first, then bring real feedback back
- provide a complete sample system, import examples, distillation examples, and local runner scripts

This release explicitly avoids multi-character theater and stays focused on one high-fidelity counselor system.
