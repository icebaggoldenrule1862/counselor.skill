# v1.0.0

`辅导员.skill` 的首个公开版本。

这一版已经把项目的主方向收住了：  
不做多角色群像，不做请假模板库，而是专注做一个可蒸馏、可纠偏、可记忆、可进化的单一辅导员系统。

## 本版包含

- 从主观描述、私聊、通知、约谈转写、制度文本中蒸馏辅导员人格
- 开口、追问、办公室约谈、说明文、补救、求助、复盘等多种模式
- correction、session、archive、snapshot 等长期层
- 现实同步路径：先预演，再把真实反馈带回来
- 完整系统样例、导入样例、蒸馏样例和本地运行脚本
- 中英文首页

## 这一版想解决什么

很多大学生不是完全不会沟通，  
而是卡在最具体的那一步：

- 第一条消息不知道怎么发
- 对方回得很冷，不知道下一句怎么接
- 明天要去办公室，脑子一片空白
- 想求助，但怕被当成找借口
- 说明文不会写，越写越像模板

`辅导员.skill` 想做的，就是在真正开口之前，先给人一个可以预演的地方。

## 使用方式

1. 描述你的辅导员
2. 导入私聊、通知、约谈、批注、制度等素材
3. 生成第一版人格
4. 开始预演真实沟通
5. 如果不像，直接纠偏
6. 如果现实里已经用了，再把真实反馈带回来

## 项目边界

这个项目用于：

- 校园沟通预演
- 情况说明起草
- 办公室约谈排练
- 真实对话复盘

不用于：

- 伪造请假材料
- 绕过学校管理
- 针对真实老师的骚扰、报复或恶意传播
