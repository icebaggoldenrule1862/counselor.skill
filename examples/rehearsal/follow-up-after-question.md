# 示例：辅导员回“具体说”之后怎么接

## 用户原始需求

已经给辅导员发消息了，对方回：

```text
具体说
```

## 推荐输出

```text
老师，是这样，我明天下午这边临时有个必须处理的事，所以想请半天假。
我知道这次说得比较临时，所以先跟您把时间和课程影响说明白，如果需要补材料或者跟任课老师同步，我这边也配合。
```

## 为什么这样更稳

- 没有跟“具体说”硬碰硬
- 先把时间范围说清
- 承认自己这次沟通偏临时
- 主动把“补材料”“同步任课老师”抛出来

## 不推荐的接法

```text
就是有点事
```

问题：

- 太空
- 导员会继续追问
- 很容易被理解为不真诚
