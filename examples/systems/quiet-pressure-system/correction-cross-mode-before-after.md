# Correction 跨模式前后对比

这份对比专门展示：

> 一次 correction，不只是把某句台词改掉，而是会同时改变多个模式。

使用的 correction 是：

- “周老师不会一开场就把话说死，更倾向先听解释，再逐步亮出老师和班长反馈，家长联动只会在后段出现。”

## 模式 1：开口器

### Correction 前

```text
老师：
你先把话说清楚，这种事情不要每次都等到最后来通知我。
```

### Correction 后

```text
老师：
你先把情况说完整，我先听你这边怎么讲。
时间线和会影响到的部分先说清楚。
```

### 差异

- 从“上来先定性”改成“先听解释再判断”

## 模式 2：追问应对器

### Correction 前

```text
老师：
具体说。
如果还是这种模糊说法，后面我就只能按流程走了。
```

### Correction 后

```text
老师：
你先把时间点和经过说清楚。
我先听你这边怎么解释，再看后面要不要补老师那边的反馈。
```

### 差异

- 追问仍然有压力
- 但先追时间线，再抬证据，不是一上来就封死

## 模式 3：办公室推演器

### Correction 前

```text
周老师：
你先别解释了，这件事我都知道了。
老师和班长那边都说过了，你现在再说意义不大。
```

### Correction 后

```text
周老师：
你先别急，先把那天的时间线说清楚。
任课老师那边已经提到你这节课没到，我先听你这边怎么解释，再决定后面怎么记。
```

### 差异

- 证据从“一次亮满”改成“先老师反馈，再班长补强”
- 第一轮依然有压迫感，但更像温和拿捏型

## 模式 4：复盘器

### Correction 前

```text
你一开始就被压住了，因为辅导员第一轮已经把话说死，后面基本没有回旋。
```

### Correction 后

```text
真正开始转硬的是第二轮以后。
第一轮其实还是给了你解释空间，但你在时间线这里没有接稳，所以后面老师和班长的反馈才逐步把压力抬高。
```

### 差异

- 复盘重点从“第一轮就完了”变成“哪一轮开始变硬”
- 这会直接影响用户下一次怎么准备

## 结论

这类对比证明：

- correction 应优先改行为节点
- 一次 correction 会同时改写开口、追问、预演、复盘
- 只有这样，它才算真正进入系统进化层
