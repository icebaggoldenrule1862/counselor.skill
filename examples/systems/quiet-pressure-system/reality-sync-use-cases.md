# 现实同步样例索引

这份索引专门回答：

> 蒸馏出自己的辅导员之后，这套系统到底能拿去现实里练哪些东西？

## 当前样例

### 1. 请假开口

- [reality-sync-leave-demo.md](./reality-sync-leave-demo.md)

适合看：

- 第一条消息怎么排练
- 为什么蒸馏后会比模板更敢发

### 2. 被追问后的接话

- [reality-sync-follow-up-demo.md](./reality-sync-follow-up-demo.md)

适合看：

- 真实聊天已经开始后，系统怎么给下一句
- 为什么“稳住当前这一轮”比讲大道理更重要

### 3. 求助型沟通

- [reality-sync-help-demo.md](./reality-sync-help-demo.md)

适合看：

- 不是违规，而是状态差、压力大、不知道怎么开口时
- 系统怎么把求助说得既不悬浮，也不显得像借口

### 4. 情况说明 / 补材料

- [reality-sync-statement-demo.md](./reality-sync-statement-demo.md)

适合看：

- 说明文结构怎么起
- 哪些地方最容易被打回

## 这套索引想说明什么

请假只是冰山一角。
真正成熟的单一辅导员系统，应该能覆盖很多现实里更有压力、也更难开口的沟通节点。

所以这些样例不是题材库，而是在展示：

> 同一个蒸馏后的辅导员系统，如何在不同现实用途下依然保持像同一个人。
