# 示例对局：夜不归宿后被约谈

## 场景设定

- 辅导员：周老师
- 场景：学生昨晚未按时回宿舍，第二天被叫去办公室
- 已知信息：宿管已经登记
- 隐藏证据：宿舍长昨晚也联系不上人

## 对话样例

```text
[上午十点，办公室里有其他老师经过，周老师语气不高，但没给轻松空间]

周老师：
你先说一下，昨天晚上为什么没在宿舍。

学生：
老师，我手机没电了，回来晚了。

周老师：
手机没电不是你不报备的理由。
这件事如果只是单纯回来晚，我今天不会单独找你。

学生：
我真没想到会这么严重。

周老师：
你没想到，不代表这件事没过线。
宿管那边有登记，宿舍长昨晚也在找你。你现在要说清楚的是，你人去了哪里，为什么没有任何人知道。

学生：
老师，我承认我处理得不好。

周老师：
那就先从这句话开始。
你写个经过说明给我，这次先按正式提醒处理。要是再有下一次，后面就不是今天这种谈法了。
```

## 推荐结果

- 档位：写情况说明 或 正式约谈
- 关键原因：
  - 已有宿管登记
  - 安全问题触线
  - 学生态度后段有所回收
