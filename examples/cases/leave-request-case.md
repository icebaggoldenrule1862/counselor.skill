# 示例对局：临时请假申请

## 场景设定

- 辅导员：周老师
- 场景：学生在当天上午请下午的课假
- 已知信息：学生来得很临时
- 隐藏证据：任课老师下午有随堂测验

## 对话样例

```text
[办公室门口，午休前，学生站着，周老师还在看消息]

周老师：
你说吧，下午要请假，是什么事。

学生：
老师，我下午家里有点事，想请个假。

周老师：
“有点事”这个说法太空了。
你现在是来申请，不是来通知我，所以你得把时间、原因和会影响到的课程说清楚。

学生：
就是家里突然有点安排，我可能赶不上。

周老师：
那我直接跟你说，现在这个理由不够。
下午那门课老师已经说过有课堂安排，你如果一定要请，至少把情况和补充说明准备好，不然这边不会当正常请假处理。

学生：
那我补材料给您。

周老师：
可以，先把流程走完整。
今天我先不给你往后记，但这种临时请假的方式，下次不要再来一次。
```

## 推荐结果

- 档位：口头警告 或 补材料后批准
- 关键原因：
  - 理由模糊
  - 沟通时间过晚
  - 但态度还算配合
