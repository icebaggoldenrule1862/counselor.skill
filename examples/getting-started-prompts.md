# 第一次使用可以直接复制的 Prompt

下面这些 prompt 的目标是让用户不需要先理解整个系统，也能立刻开始。

## 1. 开口前排练

```text
我明天想找辅导员请假，但这个理由不太好说，你先帮我排练第一句。
```

## 2. 被追问后的应对

```text
我已经给辅导员发消息了，他回我“具体说”，后面我怎么接比较稳？
```

## 3. 约谈前预演

```text
明天导员要找我谈逃课，你按最真实的方式跟我预演一下，别太温柔。
```

## 4. 聊崩后的补救

```text
我刚刚给辅导员回得有点冲，怕事情升级，帮我想一条补救消息。
```

## 5. 情况说明写作

```text
我要写一份情况说明，别写得太模板，像真实大学生会发给辅导员的那种。
```

## 6. 求助型沟通

```text
我不是想骗假，我是真的最近状态不太行，但不知道怎么跟辅导员说，你帮我先排练一下。
```

## 7. 争取通融

```text
这件事严格说不算完全合规，但我想争取一次通融，你帮我想一个不让辅导员反感的说法。
```
